# 路由

`next-intl` 在两个地方集成了 Next.js 的路由系统：

1. **代理 / 中间件**：协商语言环境并处理重定向和重写（例如 `/` → `/en`）
2. **导航 API**：对 Next.js 的导航 API（如 `<Link />`）的轻量封装

这使您可以通过类似 `<Link href="/about">` 的 API 来表达您的应用，同时语言环境和用户面向的路径名等方面会在幕后自动处理（例如 `/de/über-uns`）。
